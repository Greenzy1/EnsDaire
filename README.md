# EnsDaire - Professional Circle Mini-Game Plugin (v2.0)

EnsDaire adds a premium gaming experience to your server with advanced features, modernized code structure, and a completely customizable design. In version 2.0, the infrastructure has been fully revamped and performance-optimized.

## 🌟 Key Features

*   **Modernized Architecture (v2.0):** Code structure moved to the `org.byauth` package for a more modular and extensible system.
*   **GUI-Driven Management:** All player and admin actions are handled through modern, configurable interfaces (found in the `menus/` folder).
*   **Advanced Cosmetics:** Victory effects, custom shop system, and BCoin integration.
*   **Dynamic Scoreboard & Holograms:** Customizable, performance-friendly scoreboards and `TextDisplay`-based leaderboards.
*   **Flexible Configuration:** Control every detail via `config.yml`, `guis.yml`, `cosmetics.yml`, and detailed language files.
*   **Advanced Arena Control:** Easy setup menus, automatic arena generation, and advanced lobby systems.

## 🛠️ Commands

| Command | Aliases | Description |
| :--- | :--- | :--- |
| `/ensdaire` | `/ed`, `/daire` | Main admin command and management panel. |
| `/join` | - | Quickly joins an available arena. |
| `/leave` | - | Leaves the active arena. |
| `/cosmetic` | `/market` | Opens the Cosmetics/Shop menu. |
| `/support` | - | Supports players while in spectator mode. |

## ⚙️ Renewed File Structure

*   `config.yml`: Global game settings (durations, lobby settings, currency system).
*   `menus/`: Directory for GUI designs (admin panel, arena management, etc.).
*   `lang/`: Multi-language support (Default: `messages_tr.yml`).
*   `arenas/`: Data directory where created arenas are stored.

## 🚀 Setup & Update

1.  Place the `EnsDaire-2.0.jar` file into the `plugins` folder.
2.  Dependencies: `Paper 1.20.1+`, `PlaceholderAPI`, `Vault`, `LuckPerms`.
3.  Start the server and use `/ensdaire setlobby <arena>` to define spawn points.
