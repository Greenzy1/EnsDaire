# EnsDaire - Professional Circle Mini-Game Plugin

A professional, high-performance, and fully customizable Minecraft minigame plugin.

## 🌟 Key Features

*   **GUI-Driven Management:** All player and admin operations are handled through modern, configurable interfaces.
*   **Advanced Cosmetics:** Kill effects, victory effects, and more included in an advanced cosmetic system.
*   **Dynamic BossBar & Actionbar:** Animated indicators tracking game status (countdown, time, survivors).
*   **Animated Scoreboard:** Customizable, animated titles and comprehensive info tables.
*   **Holographic Leaderboard:** Sleek, premium-looking leadership boards showing top 10 players.
*   **Data-Driven:** Everything from GUI titles and slots to item lore and materials is changeable via YAML files.
*   **Modern Codebase:** Advanced API, NamespacedKey-based item recognition, and performance-oriented architecture.

## 🛠️ Commands

*   `/daire` - Opens the main game menu (Arena Selection).
*   `/katil <arena>` - Quickly joins the specified arena.
*   `/ayril` - Leaves the current game (restores state data).
*   `/kozmetik` - Opens the cosmetic shop.
*   `/istatistik` - Shows your personal stats.
*   `/ed admin` - Opens the management panel.
*   `/ed reload` - Instantly reloads all files.

## ⚙️ Configuration Files

*   `config.yml`: Global game settings, timings, and mechanics.
*   `guis.yml`: Design, titles, and items for all interfaces (Menus).
*   `cosmetics.yml`: Cosmetic categories and content.
*   `messages_en.yml`: Configuration for all messages and prefixes in English.

## 🚀 Installation

1.  Drop the JAR file into the `plugins` folder.
2.  Start the server and edit the generated files to your liking.
3.  Apply your settings with the `/ed reload` command.
